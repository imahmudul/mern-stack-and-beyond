
## Description

MERN & Beyond - TypeScript starter repository.

## Installation & Running the backend

```bash
$ cd backend
$ yarn install

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running the frontend

```bash
$ cd frontend
$ yarn install

# development
$ yarn run start

```

## Running with Docker
```bash
$ docker-compose up -d
```
